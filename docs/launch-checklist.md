# Launch Checklist

Use this checklist before promoting the repo publicly.

## GitHub surface

- [ ] README has a clear tagline and copy-paste prompt.
- [ ] GitHub description is concise and benefit-led.
- [ ] Topics are set: `ai-agents`, `job-search`, `resume`, `career`, `cover-letter`, `workflow`, `agentic-ai`, `templates`, `automation`, `ats`, `local-first`.
- [ ] CI badge is green.
- [ ] Issues and PR templates are present.
- [ ] Roadmap and contributing guide are present.

## Demo readiness

- [ ] `scripts/init_workspace.py` works from a clean clone.
- [ ] `scripts/validate_workspace.py` passes.
- [ ] Synthetic CV example is clearly fake.
- [ ] No private data or local personal paths are included.

## Promotion copy

Short positioning:

> A local-first, agent-agnostic workflow that turns a CV into a structured job-search system and draft application-pack pipeline.

Good launch channels:

- Hacker News Show HN;
- X/Twitter thread;
- LinkedIn post;
- r/LocalLLaMA;
- r/ClaudeAI;
- r/SideProject;
- relevant AI-agent Discords/Slack groups;
- awesome AI agents / LLM apps lists.
