## Hacker News / Show HN

**Title:** Show HN: Agent-agnostic job search pipeline from CV to application packs

I built an open-source, file-based workflow that lets any capable AI/coding agent turn a CV into a structured job-search workspace.

It is intentionally local-first and agent-agnostic: no SaaS signup, no specific model/provider, no auto-applying. The workflow covers CV ingestion, structured candidate profile, CV feedback, cover-letter style, search criteria, tracker, source-backed role scoring, and draft application packs with provenance.

Repo: https://github.com/RichardAtCT/career-application-pipeline

I would welcome feedback on the workflow, schemas, safety boundaries, and examples for different agent runtimes.

## LinkedIn

I’ve open-sourced a practical AI-assisted job-search workflow.

It is not another resume keyword tool and it does not auto-apply to jobs. It gives an AI assistant a structured, local-first process for helping someone move from CV to targeted applications:

- ingest a CV;
- identify role tracks;
- suggest CV and cover-letter improvements;
- build search criteria and a tracker;
- verify live roles from source URLs;
- create draft application packs for strong matches.

The repo is agent-agnostic, so it can be used with Claude, Codex, ChatGPT, Cursor, local agents, or a human assistant following the files.

GitHub: https://github.com/RichardAtCT/career-application-pipeline
